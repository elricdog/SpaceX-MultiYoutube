Chat

If chat window doesn't work for you, maybe your navigator is blocking because unsecure sll certificate.
Certificates are expansive and we don't want to pay for one, for now.
You can try to open this url: https://elricdog.homedns.org:5000 and go to more information and allow acces to this web. Later, if your web browser cahces that cecision you will be able to enter to the chat.

Cross-Origin web sources

Some sources that points to certain webs, will need you to install some plugin in your browser to "Ignore X-Frame headers".
- Cameron County SpaceX website.
- BocaChica.com website.
- Twitter direct Home.
- Reddit direct Home.
- Discord direct Home.
- Youtube direct Channels.
Those kind of sites has a mark (symbol *).
Is up to you to discover how to do it. Maybe we could help one, we can help everyone
